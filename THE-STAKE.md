# The Stake

Every project is built on assumptions.

Most remain unstated.

This document states the assumption on which Buffy is built.

It is the stake in the ground from which every architectural decision, every language feature, every implementation choice, and every future contribution should be measured.

If this assumption proves false, the project should change.

If it proves true, everything else should naturally follow.

---

## The Axiom

**Reality is primary. Everything else is derived.**

Reality exists independently of software.

Calendars do not create reality.

They attempt to describe it.

They succeed only to the extent that they are manually maintained as reality changes.

That maintenance is repetitive.

It is fragile.

It is easy to forget.

Over time, calendars drift from reality.

Buffy exists to reduce that drift—not by replacing calendars, but by continuously maintaining the highest-fidelity model of reality that can be derived from the available evidence.

---

## Therefore

If reality is primary...

...then calendars are observations, not truth.

...then availability is not entered. It is derived.

...then rules exist to describe reality, not to control software.

...then every constraint matters, whether it is entered explicitly or inferred automatically.

...then every decision Buffy makes should be explainable.

...then manual maintenance is a failure of the model, not a responsibility of the user.

Buffy's purpose is not to decide what should happen.

Its purpose is to maintain the highest-fidelity model of reality that can be derived from the available evidence, allowing people and software to make better decisions from the same shared understanding of what is possible.

Everything else is derived.

---

## The Test

Whenever a design decision is made, ask one question:

> **Does this bring Buffy closer to the highest-fidelity model of reality that can be derived from the available evidence?**

If the answer is yes, it probably belongs.

If the answer is no, it probably doesn't.

If the answer is unclear, return to **The Stake**.

Everything else is derived.
